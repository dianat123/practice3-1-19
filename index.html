Example for If Else condition

The border for this section has been set usinf if else condition of SASS

The mixin used is as follows

@mixin border-prop($width, $color, $border-style, $direction: null){ @if ($direction){ border-#{$direction}: $width $border-style $color; } else{ border: $width $border-style $color; } }
